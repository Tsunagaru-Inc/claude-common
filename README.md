# ツナガル × Claude Cowork 共通設定リポジトリ

このリポジトリは、ツナガル株式会社のメンバー全員が **Claude Cowork**（デスクトップアプリ）を
より効果的に使えるよう、共通の設定・コンテキストを管理するためのものです。

---

## このリポジトリに含まれるもの

| パス | 内容 |
|------|------|
| `CLAUDE.md` | プロジェクトの「カスタム指示」に貼り付ける共通設定 |
| `context/tsunagaru-company.md` | 会社情報・用語集・事業理解のコンテキスト |
| `setup/first-time-setup.md` | Claude に貼り付けるだけの対話型セットアップガイド |
| `setup/faq.md` | よくある質問と答え |
| `skills/brand-guideline-ppt.skill/` | ブランドガイドライン準拠のスライド作成スキル |

---

## はじめてのセットアップ

### ステップ 1 — Claude Cowork をインストール・起動

まだの場合は [claude.com/download](https://claude.com/download) からインストールしてください。

### ステップ 2 — 以下の一行を Claude に貼り付けて送信する

```
このURLのファイルを取得して、その中の指示に従ってください：https://raw.githubusercontent.com/Tsunagaru-Inc/claude-common/main/setup/first-time-setup.md
```

Claude が日本語でステップごとに案内します。設定完了まで約 5〜10 分です。

---

## 使い方のヒント

- **日本語でそのまま話しかけてください。** 「提案書を作って」「このメールを添削して」など
- **プロジェクトを作ると、毎回の説明が不要になります。** 設定済みのプロジェクトを開くだけで、ツナガルの文脈を理解した状態で始められます
- **やりたいことを自然な言葉で伝えてください。** Claude が適切な機能を自動的に選んで使います
- **わからないことは Claude に直接聞いてください。** 「このアプリで何ができるか教えて」など
