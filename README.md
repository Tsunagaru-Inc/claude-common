# ツナガル × Claude 共通設定リポジトリ

このリポジトリは、ツナガル株式会社のメンバー全員が Claude をより効果的に使えるよう、
共通の設定・コンテキストを管理するためのものです。

---

## このリポジトリに含まれるもの

| パス | 内容 |
|------|------|
| `custom-instructions.md` | プロジェクトの「カスタム指示」に貼り付ける共通設定 |
| `context/tsunagaru-company.md` | 会社情報・用語集・事業理解のコンテキスト |
| `skills/brand-guideline-ppt/` | ブランドガイドライン資料（参考用） |
| `setup/first-time-setup.md` | Claude に貼り付けるだけの対話型セットアップガイド |

---

## はじめてのセットアップ

### ステップ 1 — Claude デスクトップアプリをインストール・起動

まだの場合は [claude.com/download](https://claude.com/download) からインストールしてください。

### ステップ 2 — 以下の一行を Claude に貼り付けて送信する

```
このURLのファイルを取得して、その中の指示に従ってください：https://raw.githubusercontent.com/Tsunagaru-Inc/claude-common/main/setup/first-time-setup.md
```

Claude が日本語でステップごとに案内します。設定完了まで約 3〜5 分です。

---

## 使い方のヒント

- **日本語でそのまま話しかけてください。** 「提案書を作って」「このメールを添削して」など
- **プロジェクトを作ると、毎回の説明が不要になります。** 設定済みのプロジェクトを開くだけで、ツナガルの文脈を理解した状態で始められます
- **会話ごとに必要な背景は補足してください。** 案件名・クライアント・目的など
